# ShowWork

Portfolio profiles with built‑in themes. Public link format:

`https://showwork.in/<username>`

## Quick Start

1. Pick a username (e.g., `kanishk`) → `https://showwork.in/kanishk`.
2. Choose a theme from the gallery.
3. Add your bio, projects, and social links.
4. Share your profile.

## Features

- Built‑in themes
- Simple, shareable username URL
- Mobile‑friendly layouts
- Basic SEO and social preview
